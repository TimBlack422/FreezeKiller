## English Introduction

### Program Overview
    This is a small tool that can forcefully and without a password thaw a computer frozen by DeepFreeze.
    This utility allows you to freely enable, disable, or remove DeepFreeze on a frozen computer without requiring a password.

### System Requirements
    Due to compilation limitations, currently only supports unlocking computers running Windows 10 and above.
    Theoretically, this tool can unlock any version of DeepFreeze.

    PS: A version compatible with Windows 7 and above will be released once the project reaches 10 stars.
    An English version will be released once the project reaches 20 stars.

### Usage Instructions
    1.Disable driver signature enforcement.
    2.Run the software and follow the on-screen prompts.

### Compilation Environment
    Visual Studio 2022 + WDK10.0.26100.6584 + MFC

### Final Notes
    My driver is not digitally signed, so disabling driver signature enforcement is mandatory to use this program. I hope this may change in the future.
    You can contact me at:

    cheuitim12@gmail.com
    emailfortimsgithub@foxmail.com (preferred)

    A tool for forcibly thawing Shadow Defender and Reboot Rx will be developed once the project reaches 100 stars.
    The code might be a bit messy, but it’s fully commented, so you should be able to follow it easily.

    Happy exploring!
